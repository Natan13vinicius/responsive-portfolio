O Intense Oficial é um catálogo de produtos de bijuteria em uma loja online que visa facilitar a visualização dos produtos e, consequentemente, tornar mais fácil a captação de clientes. O site está atualmente disponível em uma plataforma de deploy, e os clientes podem acessá-lo através do link presente na rede social do Intense.

O desenvolvimento do site foi feito com HTML, CSS e JavaScript

Além disso, o site foi desenvolvido seguindo o conceito de "Mobile First", ou seja, priorizando a visualização em dispositivos móveis. Com isso, os clientes podem acessar o catálogo de produtos em qualquer dispositivo com facilidade.

Para gerenciamento de versão do código-fonte, foi utilizado o Git Flow, garantindo uma gestão mais organizada e eficiente do processo de desenvolvimento. O Bootstrap, por sua vez, foi utilizado como framework de front-end, agilizando o desenvolvimento e garantindo uma interface de usuário mais moderna e atraente.
