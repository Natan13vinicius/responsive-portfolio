Sobre Os Projetos:

U.M.I (Urologia Minimamente Invasiva)

Veja Detalhes
Html Css Seo
Scss Bootstrap
Mobile First Media query
Javascript Git

projeto feito em parceria atraves de ferramentas git, utilizando técnicas como HTML, CSS, Bootstrap, Media Query, JavaScript, Git e Versionamento de Arquivo(linhas de comando git). Além disso, o site foi hospedado na plataforma via "Custom Domain" por ferramenta de deploy (Vercel).

O site Umi foi criado para informar os clientes sobre os serviços oferecidos por dois médicos urologistas, bem como captar e facilitar o contato com os mesmos para resolver problemas de saúde relacionados. Com isso, o projeto está agregando valor à prática médica e proporcionando uma experiência positiva para os usuários

projeto realizado através da empresa 'Weshare', onde atuo como desenvolvedor web freelancer.
